Past meetings of the Baltimore Perl Mongers

Archives of Past Meetups

This page contains a list of the various meetups that have happened over the years at the Baltimore Perl Mongers. Did you give a talk, or see a really interesting one, but can't remember when or who? This page should be able to help.
The descriptions have been slightly edited to fit on this page a little better. For a complete description as it appeared for users, please use the attached Meetup.com links for the relevant meeting. To quick-jump to a year, use one of the following links:
2016 :: 2015 :: 2014 :: 2013 :: 2012 :: 2011

2016 Archive

  • April 16-17, 2016: Meetup, Location
    The annual DC-Baltimore Perl Workshop was held on the weekend of April 16th, and featured talks from many speakers. See the 2016 DCBPW page for more information.
  • March 24, 2016: Meetup, Location
    The DC-Baltimore Perl Workshop (http://www.dcbpw.org/) is next month (April 16&17). Maybe you are giving a talk there or have been toying with the idea of making a presentation somewhere else, like YAPC (http://www.yapcna.org) this June. The idea is we can do a run through of your talk with a much smaller audience and get some feedback before you go live! If not, let's get together anyway and catch up in the usual way with pizza.
